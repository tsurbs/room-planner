  • V — Furnish mode
  • E — Edit floorplan

  • Double-click label — Focus label text field

  • Sessions panel — Rejoin recent shared plans or restore local draft
  • ⌘⇧N — Clear floorplan (blank canvas)

  • ⌘Z / ⌘⇧Z — Undo / redo
  • R — Rotate 90° (or drag handle)
  • ⌘D — Duplicate selection
  • Alt+drag item — Duplicate & move
  • Shift+click — Multi-select
  • ⌫ — Delete selection
  • [ / ] — Send backward / bring forward
  • { / } — Collapse catalog / properties
  • Arrows — Nudge item 3″
  • Scroll — Zoom at cursor
  • Space / middle-drag / Alt+drag — Pan canvas
  • Shift+resize — Lock aspect ratio
  • Trace image — Upload or paste a floorplan photo behind the grid; drag and resize in Furnish mode
